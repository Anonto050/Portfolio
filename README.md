## Installation and How to Run

To set up and run this portfolio locally, follow these steps:

### Prerequisites

Ensure you have the following installed:
- Node.js
- npm or yarn

### Installation Steps

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/Anonto050/Portfolio.git
   ```

2. **Navigate to the Project Directory:**
   ```bash
   cd Portfolio
   ```

3. **Install Dependencies:**
   Using npm:
   ```bash
   npm install
   ```
   Using yarn:
   ```bash
   yarn install
   ```

### Running the Application

1. **Start the Development Server:**
   Using npm:
   ```bash
   npm run dev
   ```
   Using yarn:
   ```bash
   yarn dev
   ```

2. **Open Your Browser:**
   Visit `http://localhost:3000` to see your portfolio in action.

## Deployment on Vercel

Deploying this portfolio on Vercel is straightforward. Follow these steps:

1. **Create a Vercel Account:**
   If you don't have an account, sign up at [Vercel](https://vercel.com/).

2. **Import GitHub Repository:**
   - Go to the Vercel dashboard.
   - Click on "New Project" and import your GitHub repository.
   - Vercel will automatically detect the project settings.

3. **Configure Build Settings:**
   - Make sure the build command is `next build`.
   - Set the output directory to `.next`.

4. **Deploy:**
   Click "Deploy" and Vercel will build and deploy your project. Once the deployment is complete, you will receive a live URL for your portfolio.

5. **Custom Domain (Optional):**
   If you want to use a custom domain, you can add it in the Vercel dashboard under the "Domains" section.
